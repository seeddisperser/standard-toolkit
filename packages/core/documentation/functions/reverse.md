# Function: reverse()

> **reverse**\<`T`\>(`arr`): `T`[]

Returns a new array with the order of the elements reversed.

## Type Parameters

• **T**

## Parameters

• **arr**: `T`[]

## Example

// [5, 4, 3, 2, 1]
